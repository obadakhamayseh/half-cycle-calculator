# Half Cycle Calculator — Gann Square of 9

تطبيق حاسبة نصف الدورة (180°) لمربع التسعة لـ Gann.

## المعادلة
- **HIGH** → `(√Swing − Factor × n)²`
- **LOW**  → `(√Swing + Factor × n)²`

## تشغيل على Windows
```bash
pip install kivy
python main.py
```

## بناء APK للأندرويد عبر GitHub
1. ارفع المشروع على GitHub
2. اذهب إلى **Actions** → **Build Android APK**
3. اضغط **Run workflow**
4. انتظر ~15 دقيقة ثم حمّل الـ APK من **Artifacts**
